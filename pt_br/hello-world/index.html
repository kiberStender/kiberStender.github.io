Olá meu caro

Esta é apenas uma introdução sobre o que você irá encontrar neste blog. Meu nome é Kleber Stender(meus amigos me chamam Kiber, essa é a origem do nome do Blog)

Por que escrevo?

O motivo pelo qual este bog foi criado é que uma vez que você adquire certo conhecimento em sua vida, você precisa praticá-lo frequentemente, ou então ou você o esquecerá ou pior ainda, você esquecerá parte disto tornando complicado para você provar que realmente possui tal conhecimento.(Também conhecido com entrevistas ruins)

Então para documentar tudo o que eu aprender facilitando uma futura pesquisa minha ou de qualquer outra pessoa que ache isto útil, eu irei postar minhas atividades, links, explicações e qualquer coisa que vise facilitar o entendimento de algum assunto.

O começo

Para começar, combinei com um amigo(Cesar's Blog) de fazermos um projeto juntos, siumulando uma rotina normal de uma empresa com todos os rituais de scrum: Sessões de Brainstorm para determinar o projeto como um todo, criação de MVP (Produto mínimo viável), sessões de Grooming para entender e criar tarefas, sessões de Planning para quebrar tarefas em sub-tarefas facilitando a estimativa, interações de Sprints para desenvolver as tarefas, sessões de retrospectiva para saber o que foi bom e o que foi ruim, destacando o que foi bom para continuar igual o melhorar e o que foi ruim para entender porque foi ruim e para que não se repitam os mesmos erros. Este projeto é chamado Projeto Portfel(Portfel significa carteira em polonês). Ele irá postar no Blog dele seu modo de ver e pesar, enquanto eu irei postar as mesmas coisas aqui mas com meu ponto de vista e modo de pensar.

Além do Projeto Portfel ireo postar outras coisas como meus estudos de programação funcional e qualquer outra coisa relacioanda a programação que eu me sinta inclinado a postar ou que eu ache que será util a mim ou alguém de algum jeito.

Este blog é escrito em duas linguagens por agora: Inglês e Hue hue Br Br. No futuro pretendo escrêve-lo em todos os idiomas que eu souber falar usando o Blog como uma ferramenta para prática-los.
